## Quickstart

Siga estes passos para realizar sua primeira transação de teste.

<Steps>
  <Step title="Obtenha suas Chaves de API">
    Acesse o [Dashboard](https://app.pagueoneip.com.br/integrations/api) e gere suas credenciais.
    
    Você precisará da **Public Key** (`x-api-key`) e da **Secret Key** (`x-api-secret`).
  </Step>
  <Step title="Autentique sua Requisição">
    Adicione as chaves no header de todas as suas chamadas.
    
    ```bash
    x-api-key: pk_live_...
    x-api-secret: sk_live_...
    ```
  </Step>
  <Step title="Crie uma Cobrança Pix">
    Envie um POST para `/payin` com os dados mínimos.
    
    ```bash
    curl --request POST \
      --url https://api.pagueoneip.com.br/payin \
      --header 'x-api-key: SUA_CHAVE' \
      --header 'x-api-secret: SEU_SEGREDO' \
      --data '{"amount": 1000, "paymentMethod": "PIX", ...}'
    ```
  </Step>
</Steps>
