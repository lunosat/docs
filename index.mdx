Nossa API permite processar pagamentos de forma simples, segura e escalável. Suportamos múltiplos métodos de pagamento, incluindo PIX, Cartão de Crédito e Boleto.

## Autenticação

Todas as requisições API devem ser autenticadas usando chaves de API. Você pode encontrar suas chaves no dashboard da sua conta.

<Warning>
  Nunca exponha seu `x-api-secret` em aplicações client-side (frontend). Ele deve ser usado apenas em comunicações server-to-server.
</Warning>

### Headers Obrigatórios

Para autenticar suas requisições, inclua os seguintes headers em cada chamada:

<ParamField header="x-api-key" type="string" required>
  Sua chave pública. Usada para identificar sua conta.
</ParamField>

<ParamField header="x-api-secret" type="string" required>
  Seu segredo de API. Usado para assinar e autorizar as requisições.
</ParamField>
