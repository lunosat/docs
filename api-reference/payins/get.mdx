---
title: 'Consultar Cobrança'
api: 'GET /payin/{id}'
description: 'Busque os detalhes de uma transação específica.'
---

Recupere todas as informações de uma cobrança, incluindo status atualizado, taxas e valores líquidos.

## Parâmetros

<ParamField path="id" type="string" required>
  Identificador da cobrança.
  
  Pode ser:
  - Nosso ID interno (`uuid`)
  - O ID externo do adquirente (`externalId`)
  - Seu ID de referência (`referenceId`)
</ParamField>

## Resposta
