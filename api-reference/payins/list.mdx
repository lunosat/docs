---
title: 'Listar Cobranças'
api: 'GET /payin'
description: 'Liste todas as cobranças da sua conta.'
---

Retorna um array com as cobranças criadas pela sua empresa, ordenadas das mais recentes para as mais antigas.

<Note>
  A listagem é filtrada automaticamente pela Chave de API utilizada na requisição. Você só verá cobranças pertencentes à sua conta.
</Note>

## Resposta

```json 200 OK
[
```
