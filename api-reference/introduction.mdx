## Bem-vindo

Esta API foi construída seguindo os padrões **RESTful**, o que significa que ela possui endpoints previsíveis, usa códigos de resposta HTTP padrão e aceita/retorna JSON.

<Note>
  Todas as requisições, sem exceção, devem conter o header `Content-Type: application/json`.
</Note>

---

## Autenticação

A segurança é nossa prioridade. Utilizamos um sistema de **duplo segredo** nos headers para garantir que apenas sua aplicação possa se comunicar conosco.

<Warning>
  **Importante:** Essa API deve ser consumida exclusivamente **Server-to-Server**. Requisições vindas de navegadores (browsers) serão bloqueadas automaticamente.
</Warning>

Para se autenticar, você deve incluir as seguintes chaves no *Header* de cada requisição:

<ParamField header="x-api-key" type="string" required>
  Sua chave **Pública**. Identifica sua empresa.
</ParamField>

<ParamField header="x-api-secret" type="string" required>
  Sua chave **Secreta**. Nunca a exponha no frontend.
</ParamField>

### Exemplo de Header

<CodeGroup>
```bash cURL
curl --request GET \
  --url https://api.pagueoneip.com.br/payin \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: pk_live_123456789' \
  --header 'x-api-secret: sk_live_987654321'
```

```javascript Node.js
const headers = {
  'Content-Type': 'application/json',
  'x-api-key': 'pk_live_123456789',
  'x-api-secret': 'sk_live_987654321'
};
```
</CodeGroup>
