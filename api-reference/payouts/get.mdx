---
title: 'Obter Payout'
api: 'GET https://api.legacyecom.com.br/payout/{id}'
description: 'Resgata as informações totais em metadados de um único evento de saque pelo seu código único de Banco de Dados.'
---

Você pode enviar uma pesquisa sobre um determinado ID caso queira recuperar o recibo transacional da instituição processante do saque em questão, e salvar em seu log.

## Path Parameters
<ParamField path="id" type="string" required>
  ID gerado na construção através da rota [Criar Payout](/api-reference/payouts/create).
</ParamField>

## O Retorno Detalhado

O endpoint devolverá o objeto JSON:

```json
{
  "id": "payout_XXX299",
  "status": "COMPLETED",
  "amount": 250000,
  "pixKey": "test@legacyecom.com.br",
  "pixKeyType": "EMAIL",
  "referenceId": "comissionado_002",
  "beneficiaryName": "Pessoa Destino Exemplo",
  "bankReceipt": {
    "endToEndId": "E0000000020260302...",
    "transactionId": "...",
    "processedAt": "2026-03-02T16:05:00.000Z"
  },
  "createdAt": "2026-03-02T16:00:20.000Z"
}
```
