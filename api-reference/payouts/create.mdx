---
title: 'Solicitar Saque'
api: 'POST /payout'
description: 'Realize transferências Pix instantâneas.'
---

Permite enviar valores da sua conta para qualquer chave Pix de terceiros ou conta bancária.

## Validadores e Saldo

<Warning>
  Se o saldo disponível for menor que o valor solicitado, a API retornará o erro `Insufficient balance` (400).
</Warning>

## Body

<ParamField body="amount" type="integer" required>
  Valor do saque em **centavos** (inteiro).
  Exemplo: R$ 50,00 deve ser enviado como `5000`.
</ParamField>

<ParamField body="pixKey" type="string" required>
  A chave Pix de destino para onde o dinheiro será enviado.
</ParamField>

<ParamField body="pixKeyType" type="string" required>
  O tipo da chave Pix informada.
  Valores aceitos: `CPF`, `CNPJ`, `EMAIL`, `PHONE`, `EVP` (chave aleatória).
</ParamField>

<ParamField body="referenceId" type="string" required>
  Seu ID interno único para controle dessa transação.
</ParamField>

<ParamField body="beneficiaryName" type="string">
  Nome completo do beneficiário.
  *Recomendado enviar para validação de segurança.*
</ParamField>

<ParamField body="beneficiaryDocument" type="string">
  CPF ou CNPJ do beneficiário (somente números).
  *Recomendado enviar para validação de segurança.*
</ParamField>

<ParamField body="webhookUrl" type="string">
  URL para receber notificações de status da transferência (Webhooks).
</ParamField>

## Exemplos

<CodeGroup>
```bash cURL
curl --request POST \
  --url https://api.pagueoneip.com.br/payout \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: SUA_CHAVE_PUBLICA' \
  --header 'x-api-secret: SUA_CHAVE_SECRETA' \
  --data '{
  "amount": 5000,
  "pixKey": "user@email.com",
  "pixKeyType": "EMAIL",
  "beneficiaryName": "João da Silva",
  "beneficiaryDocument": "12345678900",
  "referenceId": "saque_001",
  "webhookUrl": "https://seu-site.com/webhook/payout"
}'
```

```json JSON (Body)
{
  "amount": 5000,
  "pixKey": "user@email.com",
  "pixKeyType": "EMAIL",
  "beneficiaryName": "João da Silva",
  "beneficiaryDocument": "12345678900",
  "referenceId": "saque_001",
  "webhookUrl": "https://seu-site.com/webhook/payout"
}
```
</CodeGroup>
