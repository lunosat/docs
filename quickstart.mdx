Para integrar a Legacy, você precisa de uma conta ativa com o status de **KYC Aprovado**.

## 1. Obtenha suas chaves de API

Acesse o Dashboard da Legacy e navegue até a seção de Integrações.
Lá você poderá gerar suas chaves de ambiente de Produção:

* `pk_live_...` (Sua chave Pública)
* `sk_live_...` (Sua chave Secreta)

<Warning>
**Guarde sua Secret Key (`sk_live_...`) com segurança!** Ela será mostrada apenas uma vez na criação.
</Warning>

## 2. Autentique-se na API

Por padrão, você fará as chamadas para a nossa API `https://api.legacy.com.br` enviando o cabeçalho HTTP de autorização.
Leia o guia de [Autenticação](/authentication) detalhado para mais exemplos.

## 3. Realize sua primeira cobrança PIX

Este é um exemplo simples de como gerar um PIX de $100,00 reais (lembre-se, os valores devem ser enviados em **centavos** - portanto, enviaremos `10000`).

```json
POST /payin
Content-Type: application/json
Authorization: Basic base64(pk_live_xxxx:sk_live_yyyy)

{
  "paymentMethod": "PIX",
  "amount": 10000,
  "referenceId": "meu-pedido-1234",
  "isPhysicalProduct": false,
  "payerIp": "200.100.50.3",
  "customer": {
    "name": "Maria Silva",
    "document": "83416281085",
    "email": "maria@email.com",
    "phone": "5511999999999",
    "address": {
      "street": "Rua Exemplo",
      "number": "123",
      "zipCode": "01000-000",
      "city": "São Paulo",
      "state": "SP"
    }
  },
  "items": [
    {
      "title": "Produto Teste",
      "quantity": 1,
      "unitPrice": 10000
    }
  ]
}
```

A resposta retornará um código QR e uma chave `emv` do Pix Copia e Cola para você exibir para seu cliente.
Você deverá ouvir as alterações de status através de nossos [Webhooks](/webhooks)!
